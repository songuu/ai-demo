### 状态指示器

| 状态 | 颜色 | 含义 | 可以输入？ |
|-----|------|------|----------|
| Running... | 🟢 绿色 | 进程正在运行 | ✅ 是 |
| Completed | ⚪ 灰色 | 进程已完成 | ❌ 否 |
| Failed | 🔴 红色 | 进程失败 | ❌ 否 |

| ▶️ | 启动/重启进程 |

## 🚨 常见问题 3 秒解决

### ❓ 输入没有反应？

**检查状态栏**:
- 🟢 Running → 正常，继续输入
- ⚪ Completed → 点击 ▶️ 重启
- 🔴 Failed → 查看红色错误，修复后点击 ▶️

### ❓ 没看到启动提示？

**等待更长时间**:
- 通常需要 3-5 秒
- 如果 10 秒后仍未出现，查看是否有红色错误
- 可能是 CLI 工具未安装或工作目录不对

### ❓ 看到 "No messages returned" 错误？

**解决方案**:
```
1. 查看 "WORKING DIRECTORY" 字段
2. 确保该目录存在
3. 在外部终端测试: cd <目录> && claude --continue
4. 如果外部可以，应用内也应该可以
5. 查阅 TROUBLESHOOTING.md
```

## 💡 专业提示

### 提示 1: 多会话管理
```
打开多个会话:
1. Resume 会话 A
2. 进行对话
3. 选择会话 B
4. Resume 会话 B
5. 快速切换回会话 A
```

### 提示 2: 保存输出
```
长对话完成后:
1. 点击 📄 按钮
2. 所有输出复制到剪贴板
3. 粘贴到笔记或文档保存
```

### 提示 3: 查看历史
```
对话中想查看之前说了什么:
1. 点击 "Conversation" 按钮
2. 查看完整对话历史
3. 使用搜索过滤
4. 点击 "Terminal" 返回继续对话
```

### 提示 4: 内置命令
```
进程未运行时:
- help      → 查看可用命令

- cd <dir>  → 切换目录
- clear     → 清空屏幕
```

## 🎓 进阶技巧

### 技巧 1: 快速切换视图

```
Conversation ←→ Terminal

查看历史    ←→    实时对话
搜索过滤    ←→    输入新消息
复制消息    ←→    继续交互
```

### 技巧 2: 多轮对话最佳实践

```
1. 一个问题发送一条消息
2. 等待完整响应后再发下一条
3. 如果响应未完成，等待状态回到 Running
4. 大量输出时可能需要几秒显示完
```

### 技巧 3: 错误恢复

```
出错时:
1. 不要慌张
2. 查看红色错误消息
3. 理解错误原因
4. 查阅 TROUBLESHOOTING.md
5. 修复后点击 ▶️ 重启
```
